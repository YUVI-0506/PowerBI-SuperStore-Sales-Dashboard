# Power BI Super Store Sales Dashboard

This project is a Sales Analysis Dashboard created using Microsoft Power BI.

## Features

- Sales Analysis by Region
- Sales by Category and Sub Category
- Monthly Sales Trend
- Profit Analysis
- Sales by Ship Mode

## Tools Used

- Microsoft Power BI
- Data Visualization
- Super Store Dataset

## Dataset

The dataset used in this project is the Super Store Sales dataset which contains information about:

- Orders
- Sales
- Profit
- Ship Mode
- Category and Sub Category
- Regional Sales Performance
